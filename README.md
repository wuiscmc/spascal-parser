spascal-parser
==============

Semantic parser for a pascal based programming language

Requirements

- Flex 2.3
- Byacc (Berkley Yacc) 9/9/90
- GCC 4.6.2 (Mingw)
- Windows XP SP3

Run

- Download the project 
- Place the binary (exe) files in bin/
- Modify test/doomtest 
- Execute the launcher.bat script
